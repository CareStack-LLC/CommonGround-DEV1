Understanding ARIA

ARIA (AI Relationship Intelligence Assistant) monitors messages for potential conflict and suggests more constructive rewrites. When a message is flagged:

  • Accepted - Parent used ARIA's suggested rewrite
  • Modified - Parent edited the suggestion before sending
  • Rejected - Parent sent original message unchanged
